# Lagom Streamlit Dashboard

This dashboard visualizes the final notebook outputs for:

- Opportunity Area #1: Absorption Rate Intelligence
- Opportunity Area #2: Geographic Market Prioritization

## How to use

1. Paste and run `dashboard_export_cell.py` at the end of your completed notebook.
2. Confirm a folder called `dashboard_data` is created.
3. Place `streamlit_app.py` in the same folder level as `dashboard_data`.
4. Run:

```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```

## Main dashboard pages

- Executive Summary
- OA1: Absorption Intelligence
- OA2: Geographic Prioritization
- Model Validation
- Market Drilldown
